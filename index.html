What is Boop House? Stream Automation Tool for Content Creators

Boop House is a powerful live streaming automation platform that automatically generates clips from livestreams using advanced pixel motion detection algorithms. Designed specifically for content creators, streamers, and video editors who need to streamline clip farming workflows and automate content production.

The system seamlessly connects to OBS Studio (Open Broadcaster Software) via WebSocket API to monitor livestreams in real-time. It uses intelligent computer vision techniques to detect moments of high pixel motion variation, which typically indicate engaging or interesting content moments. When motion exceeds configurable thresholds, the system automatically begins recording clips. Recording stops when motion variation settles below the threshold, ensuring you capture the full action sequence.

This automated clip generation process eliminates manual monitoring and editing, allowing content creators to focus on streaming while Boop House handles highlight detection and capture automatically.

How Boop House Works - Automatic Clip Generation Process

  1. OBS WebSocket Connection: Boop House establishes a secure connection to OBS Studio via WebSocket API, enabling real-time communication and stream access without screen capture overhead.
  2. Real-time Frame Analysis: The system continuously captures stream frames from OBS and analyzes pixel motion variation using advanced computer vision algorithms, comparing consecutive frames to detect motion patterns.
  3. Motion Threshold Detection: When pixel motion variation exceeds the configured high threshold (indicating engaging or interesting content), the automated recording system immediately triggers clip capture.
  4. Intelligent Clip Recording: Recording continues until motion variation settles below the low threshold, ensuring complete capture of action sequences and engaging moments for video editing.
  5. Metadata-Rich Output: Clips are saved with comprehensive metadata including timestamps, motion metrics, duration, and file information, ready for post-production workflows and video editing software.

Use Cases: Perfect for gaming streams, tutorial content, live events, reaction videos, and any livestream where automatic highlight detection and clip farming can save hours of manual editing time.

Quick Start Guide - Install and Use Boop House

Prerequisites: Python 3.8+, OBS Studio 28.0+, OBS WebSocket Plugin, FFmpeg

Step 1: Install Python dependencies:

pip install -r requirements.txt

Step 2: Configure environment variables. Copy .env.example to .env and set OBS WebSocket connection details (host, port, password) and motion detection thresholds.

Step 3: Launch OBS Studio, start your stream or recording, and ensure OBS WebSocket is enabled in settings.

Step 4: Run Boop House to begin automatic clip generation:

python boop_house.py

The system will automatically connect to OBS, monitor your stream for motion events, and generate clips in the configured output directory.

Advanced Configuration: Adjust motion thresholds, clip duration limits, and output settings in the .env configuration file to match your streaming style and content needs.

Technical Details - How Motion Detection Works

Boop House uses advanced computer vision techniques including frame differencing and background subtraction (MOG2 algorithm) to analyze pixel motion. The system samples frames at configurable rates (default 30 FPS), calculates motion variation percentages, and compares against high/low thresholds. Smoothing algorithms reduce false positives from camera shake or encoding artifacts. Motion must remain below the low threshold for a configurable settle duration before stopping recording, ensuring complete capture of action sequences.
